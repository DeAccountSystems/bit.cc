# bit.cc

> bit.cc

#### dev
``` bash
# install dependencies
$ npm install

# generate the i18n file
$ npm run i18n

# serve with hot reload at localhost:18888
$ npm run dev
```

### build
```bash
# build for production and launch server
$ npm run build
$ npm run start
```

### deploy
```bash 
# we use pm2 in production
$ npm run reload_production
$ npm run reload_test1
```
